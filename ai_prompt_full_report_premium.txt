## PREMIUM REPORT REQUIREMENTS

### 1. Depth Over Brevity
- Every section should provide 2-3x more detail than basic report
- Include specific evidence from their answers
- Provide root cause analysis, not just identification of issues
- Offer phased growth pathways (30/60/90 day or 3-phase structure)

### 2. Conversation Guide
- Must include at least 3 session guides
- Each session has theme, duration, key points, specific questions
- Include talking points mentor can use verbatim
- End each session with prayer suggestion

### 3. Biblical Integration
- Every major section should reference relevant Scripture
- Explain WHY the Scripture applies (don't just cite it)
- Connect Old and New Testament themes where relevant

### 4. Progress Tracking
- Compare to previous assessments if available
- Set measurable milestones with verification methods
- Recommend reassessment timeline

### 5. Mentor Support
- Include preparation notes for sensitive topics
- Note cultural/contextual considerations
- Provide prayer focus specific to this apprentice

### 6. Personalized Resources
- Resources must include "why specifically for them" explanation
- Include "how to use" instructions
- Provide conversation starters from each resource

## RULES
1. **MC = Objective**: Use correct/incorrect from ground truth for biblical knowledge
2. **Open = Insight**: Deep analysis of spiritual maturity markers
3. **Health score**: Same formula as basic (MC 60%, open 40%)
4. **Word count**: Premium reports should be 3000-5000 words equivalent in JSON content
5. **No generic advice**: Everything must reference their specific answers/situation
6. **Include previous assessment comparison** if previous_assessments data provided
7. **Return valid JSON only**. No markdown outside JSON structure.
